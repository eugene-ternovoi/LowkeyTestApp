# Make an image feed using Pexels API

https://www.pexels.com/api/documentation/#photos-curated

## Requirements:

- Cell must contain: author's name, shadow and corner radius ✅
- Pagination ✅
- On cell selection show detail page with full image ✅

## Optional:

- Add cache ✅
- Pull to refresh ✅
